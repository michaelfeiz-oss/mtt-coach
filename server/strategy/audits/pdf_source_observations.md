## CRITICAL FINDING: K2s/Q2s/J2s/T2s in UTG RFI

The "wheel suited" hands (K2s, Q2s, J2s, T2s) ARE present in the source PDFs for UTG RFI at ALL stack depths (15bb, 25bb, 40bb).

This is a legitimate GTO strategy feature — these hands have blocker value (wheel cards block straights) and nut-flush potential. They are intentionally included in tight UTG ranges while middle-rank suited hands (K9s-K3s, Q9s-Q3s etc.) are folded.

**CONCLUSION: The K2s/Q2s/J2s/T2s RAISE in UTG RFI is CORRECT per source. This is NOT a data error.**

The previous "P0" report was incorrect about this specific point. The charts are showing the correct GTO strategy.
