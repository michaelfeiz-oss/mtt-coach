# Range Engine Source-Truth Rebuild

Date: 2026-05-12  
Branch: `fix/master-range-ui-hardening`

## Purpose

This pass hardens the preflop study engine around one rule:

> No chart may enter any training flow unless it is exact source-backed or explicitly manually approved.

The goal is smaller but trustworthy training coverage. Study-only charts may remain visible in the viewer, but they are blocked from trainer, drill packs, weak spot recommendations, dashboard suggestions, and direct trainer URLs unless they satisfy the same trainer gate.

## Training gate

Trainer eligibility is now derived from shared source-truth metadata:

```ts
trainerAllowed =
  sourceStatus === "source_backed" ||
  manuallyApprovedForTraining === true
```

Blocked by default:

- `simplified_population`
- `proxy`
- `unsupported`

Current manual approvals:

- `0`

## Source-status definitions

| Status | Meaning | Trainer |
|---|---|---|
| `source_backed` | Exact chart backed by reviewed source material | Allowed |
| `simplified_population` | Shared simplified family model, not exact seat-by-seat source truth | Blocked by default |
| `proxy` | Structured study proxy, not exact training data | Blocked by default |
| `unsupported` | No acceptable source mapping for training | Blocked |
